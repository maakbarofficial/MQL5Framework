# MQL5Framework

A lightweight and easy-to-use **MQL5 framework** with reusable functions for MetaTrader 5.  
Designed to simplify trading algorithm development, strategy scripting, and custom indicators.

---

## Features

- ✅ Reusable functions for common trading tasks
- ✅ Simplified order management (buy, sell, modify, close)
- ✅ Utility functions for indicators, prices, and account information
- ✅ Logging and debugging helpers
- ✅ Modular and easy to extend
